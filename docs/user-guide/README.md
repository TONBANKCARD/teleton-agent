# Teleton Agent WebUI User Guide

This guide is the user-facing handbook for operating Teleton Agent through the WebUI. It is split into parallel Russian and English versions and includes screenshots plus architecture diagrams.

The WebUI labels in the screenshots are English because the current application interface is English. Russian pages explain the same workflows in Russian and reuse the same UI captures.
